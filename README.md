# to-do-app
a simple to-do list that allows you to add tasks & check them off

## Installation

* `npm install -g knex`
* `npm install`
* Set up a local postgres database
* Create a `.env` file, using `.env.example` as a reference
* Run `knex migrate:latest && knex seed:run`
* `npm run dev-start`

## Built With
* Node.js
* Express.js
* PostgreSQL
